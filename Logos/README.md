# Logos

All files transparent PNG unless noted. Vector (SVG) not yet available — if you have the original vector source (Illustrator/Figma), export SVG versions and add them here for crisp scaling in Canva and web use.

## Files

| File | Description | Use on |
|---|---|---|
| `primary-square-black.png` | Full lockup, stacked (icon over "MANEDA STUDIOS"), black ink | Light / white backgrounds |
| `primary-square-white.png` | Full lockup, stacked, white ink | Dark backgrounds |
| `primary-horizontal-black.png` | Full lockup, horizontal (icon beside wordmark), black ink | Light backgrounds, wide formats — email signatures, letterhead, website header |
| `icon-black-on-white.png` | Icon mark only, black ink, white square fill | Square social avatars where transparency isn't supported |
| `icon-white-on-black.png` | Icon mark only, white ink, black square fill | Square social avatars, dark UI |
| `icon-white-transparent.png` | Icon mark only, white ink, transparent background | Overlay on photographs, dark video, watermarks |
| `icon-black-on-cream.png` | Icon mark only, black ink, solid cream/light fill, no transparency | Print contexts requiring a flattened file |

## Rules

- Never place the black-ink primary lockup on a dark background, or the white-ink version on a light background. Use `icon-white-transparent.png` for photo overlays regardless of the image's tone, unless the photo is very light — then use the black icon instead.
- Maintain clear space around the logo equal to the height of the icon mark on all sides. Do not crowd it against edges or other elements.
- Do not stretch, recolor, rotate, or add effects (drop shadow, outline, gradient) to any logo file.
- Instagram profile photo: use `icon-white-on-black.png` or `icon-black-on-white.png` — square social avatars require a filled background, not transparency.
- Watermarking a photo: `icon-white-transparent.png`, small scale, bottom corner, low opacity.
